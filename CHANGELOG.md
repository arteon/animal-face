# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2025-07-01

### Added

- AI-powered facial analysis using MediaPipe's 468-point face mesh
- 12 animal type matching (Dog, Cat, Fox, Bear, Rabbit, Deer, Hamster, Wolf, Penguin, Owl, Squirrel, Dinosaur)
- 10 facial ratio feature extraction (eye width/height, nose length/width, face roundness, lip thickness, jaw angle, forehead ratio, eye distance, face taper)
- Weighted Euclidean distance matching algorithm
- React 19 + TypeScript frontend with Vite
- Tailwind CSS + shadcn/ui component library
- Framer Motion animations
- Photo upload with drag-and-drop support
- Social sharing (Twitter, Facebook, KakaoTalk)
- Bilingual support (Korean / English) with auto-detection
- Global statistics dashboard
- Privacy-first architecture — no images stored
- Python 3.11 serverless backend on AWS Lambda
- PostgreSQL database for aggregate stats
- GitHub Actions CI pipeline
